## Key Takeaways

- Build **five context files once** (2-3 hours) to eliminate repeated context-setting in every AI session—saves 5+ hours weekly across all AI tools
- Context files are **plain text files (.md format)**, not technical; no coding skills required
- Use voice memo or conversational AI interview method rather than typing—generates richer, more honest output
- Context files work across **all AI platforms** (ChatGPT, Claude, Gemini, future tools) when uploaded; not Claude Code-specific
- Vague context produces vague AI suggestions; specificity with concrete numbers/dates drives actionable recommendations

## Frameworks / Models

### Five-File Context Architecture

| File | Purpose | Key Content | Refresh Cycle |
|------|---------|-------------|---------------|
| **Constitution-Context.md** | Foundation values & non-negotiables | Core principles, trade-offs between competing values, specific examples of real decisions | Annual |
| **Work-Context.md** | Role, projects, current state | Day-to-day activities, current metrics, what's working/broken, energy drivers | Quarterly |
| **Goals-Context.md** | Direction & outcomes with deadlines | Q1-Q4 specific goals with numbers and dates, success metrics, priorities | Annual (Q1 reset) |
| **PersonalLife-Context.md** | Actual constraints & rhythms | Energy patterns, daily routines, recharge methods, living situation, gaps between ideal/reality | Quarterly |
| **Clients-Context.md** (optional) | Audience definition | Who you serve, their needs, communication preferences, pain points | As-needed |

### Anti-Pattern Detection in Self-Assessment

**Generic → Specific Conversion Framework**

- "I value integrity" → "I value intellectual honesty over being right; when efficiency conflicts with quality, quality wins"
- "I'm a consultant" → "Freelance copywriter: 60% client work (B2B SaaS landing pages), 30% pitching, 10% admin"
- "Grow my business" → "Hit $8K/month by Q2 (currently $5K); measure: monthly recurring revenue"
- "I'm a morning person" → "Wake 5:30 AM, 20 minutes silence before anyone else up—sacred ritual"
- "I need downtime" → Specify activity: reading, walking, staring at wall—AI cannot infer

## Implementation Steps

### Setup Phase (30 minutes)

1. Create folder: `Desktop/AI Context Files/`
2. Choose delivery method: **Option B (conversational AI interview) recommended over voice memos**
3. Identify your AI tool (Claude, ChatGPT, Gemini—all support file uploads)

### Build Sequence (2-3 hours total)

**File #1: Constitution-Context.md (40 minutes)**
- Copy prompt template into AI chatbot
- AI asks scenario-based questions: "When X conflicts with Y, which wins?" / "Give example of choosing X over Y" / "What makes you walk away?"
- AI flags generic answers: "Everyone says integrity—what does it mean to YOU specifically?"
- Output: 1-2 paragraphs, first-person, capturing specific values with examples and non-negotiables
- Save with filename: `Constitution-Context.md`

**File #2: Work-Context.md (50 minutes)**
- Use provided prompt; focus on **actual work, not job title**
- AI probes: "Consultant doing what?" / "Strategic initiatives = what specifically?" / Current metrics: "At what number currently? Target?"
- Output: 3-4 paragraphs covering (1) role/day-to-day, (2) what's working/broken, (3) metrics and goals, (4) energy drivers
- Save with filename: `Work-Context.md`

**File #3: Goals-Context.md (40 minutes)**
- Push specificity: dates and numbers only ("soon" → actual deadline; "grow audience" → subscriber target by date)
- Q1-Q4 breakdown with concrete milestones
- Output: 2-3 paragraphs with quarterly goals, deadlines, measurable success metrics
- Set annual reminder to rebuild January 1st
- Save with filename: `Goals-Context.md`

**File #4: PersonalLife-Context.md (45 minutes)**
- Prioritize "weird stuff": specific times you wake, coffee requirements, post-8 PM uselessness, meeting aversions
- Gap analysis: describe ideal vs. current reality (e.g., "want to wake 5 AM but actually sleep until 7")
- Output: 3-4 paragraphs covering daily rhythms, energy patterns, living constraints, actual preferences
- Save with filename: `PersonalLife-Context.md`

**File #5: Clients-Context.md (optional, 20 minutes)**
- Only if serving defined customer base
- Define: who, their needs, communication preferences, pain points

### Deployment Phase

- Upload all 5 files to **ChatGPT Projects or Claude Projects** (one-time setup; persistent across sessions)
- OR upload fresh at start of each new multi-session AI project
- Future-proof: works with any AI tool that accepts file uploads

## Metrics & Thresholds

### Quality Thresholds for Context Files

| Metric | Threshold | Validation Method |
|--------|-----------|-------------------|
| **Specificity Score** | Zero generic phrases ("integrity," "excellence," "work-life balance") | Audit text for business-speak; rewrite until each claim has concrete example |
| **Quantification** | Every goal must have number + date | "Hit 500 subscribers by June 30"; "Revenue $1K/month by December" |
| **Time to Build** | 2-3 hours total for 5 files | Voice interview method saves vs. typing |
| **Refresh Cycle** | Goals file: annual (Q1 reset); others: quarterly | Set calendar reminders |
| **Time Savings Realized** | Baseline: 5+ hours saved weekly | Track AI session context length; measure reduction in repeated explanations |

## Guardrails & Anti-patterns

### Common Failures

| Anti-Pattern | Why It Fails | Correction |
|--------------|-------------|-----------|
| Typing vs. voice memo | Typing creates stilted, formal output; voice generates richer, more honest content | Use conversational AI interview (Option B); let AI ask questions while you brain-dump |
| LinkedIn-fied language | "Strategic initiatives," "drive growth," "synergize"—AI picks up on formal tone and mirrors it back generically | Write like you're explaining to a friend; include the mundane details (meeting aversion, coffee requirement) |
| Vague goals | "Grow business," "be consistent," "improve efficiency"—produce equally vague AI suggestions | Mandate numbers + dates: not "grow," but "reach $8K/month by Q2 (currently $5K)" |
| Omitting constraints | Skipping PersonalLife details (you're a night owl, hate video calls, useless after 8 PM) | Include ALL quirks and constraints; this is the "whole point"—generic advice fails without them |
| Perfection-seeking | Rewriting Constitution file 3+ times looking for "right" framing | Prioritize **honest over perfect**; quirky, specific, imperfect beats polished and generic |
| Assuming permanence | Treating all files as static; not updating Work or Goals files quarterly | Only Constitution stays relatively stable; refresh Work/Goals/PersonalLife quarterly; reset Goals annually Q1 |

### Avoidance of Scope Creep

- Files are **context, not instruction manuals** (1-4 paragraphs max per file)
- Do not attempt to document every edge case; capture patterns and primary constraints
- Resist adding system prompts or detailed rules; AI interview method extracts essence without over-engineering

## Decision Criteria

### When to Rebuild vs. Refresh

**Rebuild (replace file):**
- Annual: Constitution-Context.md (values evolve yearly)
- Annual: Goals-Context.md (each January)

**Refresh (update with new quarter):**
- Quarterly: Work-Context.md (projects and metrics change)
- Quarterly: PersonalLife-Context.md (energy and constraints shift seasonally)

**As-needed:**
- Clients-Context.md (only if audience definition changes significantly)

### Choosing Delivery Method

- **Use Conversational AI Interview (Option B) if:** you prefer speaking over typing, have 15-20 min blocks, want richer output, multitask-friendly (walking, folding laundry)
- **Use Voice Memo (Option A) if:** you're in a noisy environment or prefer recording over typing, then have AI transcribe and organize

## Templates / Examples

### Template: Constitution-Context.md (Sample Output)

*"I value intellectual honesty over being right. When a choice comes between efficiency and quality, quality wins. I won't compromise on treating people with dignity, even in strong disagreement. I walk away from projects where I'm asked to hide the truth or work with people who dismiss feedback. I've violated my own rules when chasing money early in my career—learned that fast. Now I turn down 2-3 opportunities per month because they don't align with these principles."*

### Template: Work-Context.md (Sample Output)

*"I'm a freelance copywriter. 60% client work (B2B SaaS landing pages), 30% pitching/business development, 10% admin. Currently at $5K/month; goal is $8K/month by Q2. Working with 3 retainer clients and pitching 1-2 new projects per week. What's working: retainer clients (predictable); what's broken: sales pipeline is inconsistent, I hate cold outreach. Energy: I'm energized by strategic work and client relationships; I'm depleted by administrative tasks and repetitive editing."*

### Template: Goals-Context.md (Sample Output)

*"Q1-Q2: Hit $6K/month revenue by April 30 (from $5K); onboard 1 new retainer client. Q3-Q4: Launch paid tier ($7/month) by September 1; build to 500 newsletter subscribers by June 30; hit $1K/month supplemental revenue from paid tier by December 31. Success = specific revenue numbers hit on schedule; measuring monthly recurring revenue weekly."*

### Template: PersonalLife-Context.md (Sample Output)

*"I wake at 5:30 AM; that first 20 minutes in silence (coffee, no input) is sacred—if I lose it, the whole day is off. I'm dead by 8 PM; any suggestion for evening productivity won't happen. I recharge alone (books, walks, silence); group activities deplete me even when I enjoy them. I hate video calls before 10 AM and video calls longer than 30 minutes. I work from home (no commute). The gap: I want to be more social, but I protect solitude fiercely because it's survival, not preference. Currently at peace with this."*

### AI Interview Prompt (Copy-Paste Ready)

```
# Build [FileName]-Context.md

Build my [FileName]-Context.md file: [brief purpose]

Interview me to [specific goal - capture actual vs. aspirational/push for specificity]

**Ask about:**
[5-7 concrete questions that reveal patterns, not wishes]

**Push back if I'm vague:**
[examples of generic answers + how to reframe]

**Good answers vs bad answers:**
GOOD: [specific example with numbers/behaviors]
BAD: [generic business-speak version]

**After interviewing, write [X paragraphs] in first person:**
1. [Topic 1]
2. [Topic 2]
[etc.]

Sound like [relatable reference], not [formal version].

Start with your first question.
```

## Technical Note (Non-Critical)

**.md file format:** Plain text file with `.md` extension. Benefits: lightweight, AI-trained on markdown, portable (no Microsoft formatting bloat), preserves structure (headers, lists). Opening/editing: any text editor (Notepad, TextEdit). No syntax knowledge required—type naturally, save with `.md` extension.
